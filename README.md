BUILDING
----

You'll need npm

```
npm update
npm install
npm prune
grunt
```

But most of the time 

```
grunt
```

would work just fine.

TESTING
----

I've included a ```server.py``` file that facilitates testing. Run

```
python server.py
```

And open http://localhost:8000 in your browser
